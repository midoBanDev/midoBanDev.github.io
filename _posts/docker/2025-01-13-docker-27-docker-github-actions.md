## 워크플로우(workflow)
워크플로(workflow)는 CI/CD 파이프라인에서 하나 이상의 작업을 실행할 구성 가능한 자동화된 프로세스를 말한다. 
쉽게 말해 CI/CD 파이프라인의 실제 설계도를 의미한다. 
워크플로는 `.github/workflows` 디렉터리에 YAML 파일로 정의된다.
